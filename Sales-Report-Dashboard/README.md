# 📊 Sales Report Dashboard — Power BI

## Overview
An interactive sales analytics dashboard built using Power BI, analyzing global retail sales performance across countries, product categories, and sales channels. This project was built to practice data modeling, KPI design, and interactive report development.

## 📌 Key Features
- **KPI Cards** — Total Sales, Total Profit, Average Sales, and Total Products at a glance
- **Country Slicer** — Filter performance across 30+ countries including India, USA, UK, and more
- **Year & Month Filter** — Time-based analysis for trend tracking
- **Sales by Year and Quarter** — Line chart tracking sales trends from 2012–2014
- **Profit by Product Category** — Bar chart comparing profitability across Computers, Cameras, TV & Video, Cell Phones, and more
- **Profit by City** — Donut chart breaking down profit contribution by City of each countries

---

## 📂 Dataset
- **Source:** Sample retail sales dataset (publicly available)
- **Fields:** Sales, Profit, Cost of Sales, Unit Cost, Order Qty, Product Category, Product Sub-Category, Channel, Country, Region, Manufacturer, Promotion Name, Order Date

## 💡 Insights
- Sales peaked in **2014**, showing consistent year-on-year growth from 2012
- **Computers** emerged as the highest profit-generating product category
- **Store channel** accounts for the majority of total profit
- India was selected as a sample country filter to demonstrate geo-level drill-down capability

---

## 🚀 How to Use
1. Download the `.pbix` file
2. Open in **Power BI Desktop** (free download from Microsoft)
3. Use the **country slicer** to filter by region
4. Use **Year/Month** dropdowns to explore time-based trends
